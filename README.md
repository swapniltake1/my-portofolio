# Swapnil Take - Data Engineer Portfolio

Professional React portfolio for a Data Engineer with 2.5 years of experience.

## What Changed

- Rebuilt the old fresher portfolio into a production-style data engineering profile.
- Added a professional hero section with data-platform visual treatment.
- Added KPI cards for records processed, ETL workflows, and pipeline availability.
- Added About, Technical Skills, Experience Timeline, Featured Projects, Certifications, Contact, and Resume sections.
- Removed personal-image based presentation and old Java-developer fresher copy.
- Added a public resume page at `/resume.html`.

## Tech Stack

- React 18
- React Router
- React Icons
- Create React App

## Run Locally

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

The production bundle is generated in `build/`.

## Customize

- Update project data in `src/components/WorkCardData.js`.
- Update skills in `src/components/Skills.js`.
- Update experience in `src/components/Experience.js`.
- Update contact links in `src/components/Form.js` and `src/components/Footer.js`.
- Update resume content in `public/resume.html`.
